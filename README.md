# Messaging App Backend

Backend for a real-time messaging application built using Node.js, Express, MongoDB, and JWT authentication.

## Features (so far)
- User authentication (JWT)
- One-to-one chat system
- Protected APIs
- Clean project structure

## Tech Stack
- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT

## Setup
1. Clone the repository
2. Install dependencies
   ```bash
   npm install
   ```
